## Can you list down the the topics on which questions were asked in the interviews or any other insights on the interview process?
DP, Backtracking
Question 1. Given a string s, partition s such that every substring of the partition is a palindrome.
Return the minimum cuts needed for a palindrome partitioning of s.

Sliding window, Hashing
Question 2. Given two strings s and t of lengths m and n respectively, return the minimum window substring of s such that every character
in t (including duplicates) is included in the window.
If there is no such substring, return the empty string.

Explain your answers properly, also explain your thought process.
